# Shoe_inventory

This is a simple application built with Vue and Express. It uses plain json array to store data.
It allows you to keep track of your shoe inventory. You can add, update, delete and view your shoes.

## Project setup

```
cp .env.example .env
```

```
npm install
```

### Running the app
```
npm start
```
